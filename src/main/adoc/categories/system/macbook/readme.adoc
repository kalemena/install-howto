= Macbook
:hardbreaks:

.First boot:

- Follow standard procedure of Mac install

.Setup Principal:

* System -> Apple Account Id
* System -> WallPaper
* System -> Screensaver
* System -> Trackpad
** Change “natural sliding”
* System -> Bluetooth
** Keyboards
** Trackpad
** Headphone
* Network
** DNS

.Command lines installation

[source,bash]
----
# Xcode
> xcode-select --install

# Brew
> /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Applications

> brew install rectangle
# Set rectangle to open on session start

> brew install --cask firefox

# Open “about:config”
# Set:
# - browser.sessionstore.resume_from_crash=true
# - browser.sessionstore.max_tabs_undo = 50
# - browser.sessionstore.enabled=true

> brew install --cask thunderbird
> brew install --cask gimp
> brew install --cask calibre
> brew install --cask proton-pass
> brew install --cask telegram

# VSCode
> brew install --cask visual-studio-code

# Extentions
# - Remote-SSH
# - Asciidoc

----

.Docker

[source,bash]

> brew install docker colima
> colima start --cpu 8 --memory 8 --vm-type=vz --vz-rosetta
