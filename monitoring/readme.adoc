# Monitoring docker stack

This is an aggregate of many resources out there to run some monitoring stack.

## Simple telegraf + grafana + influxdb

`docker-compose` includes `telegraf`, `grafana` and `influxdb`.

### Usage

* Start the stack

    $ docker-compose up -d
    ...
